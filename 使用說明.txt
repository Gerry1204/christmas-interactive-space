如何開啟並執行本專案

1. 安裝必要套件 (如果尚未安裝)
   請在終端機 (Terminal) 輸入以下指令：
   npm install

2. 啟動開發伺服器 (開發模式)
   這會開啟即時預覽，您可以邊改程式邊看結果。
   輸入指令：
   npm run dev
   
   成功啟動後，按住 Ctrl 並點擊終端機出現的網址 (通常是 http://localhost:5173)。

3. 建置正式版本 (部屬用)
   若要將程式打包成網頁檔案 (位於 dist 資料夾)：
   輸入指令：
   npm run build

4. 預覽正式版本
   查看打包後的結果：
   輸入指令：
   npm run preview
